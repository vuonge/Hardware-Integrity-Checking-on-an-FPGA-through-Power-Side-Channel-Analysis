# Hardware-Integrity-Checking-on-an-FPGA-through-Power-Side-Channel-Analysis
Repository for storing project code and scripts used in my MS thesis: "Hardware Integrity Checking on an FPGA through Power Side-Channel Analysis"

Usage: 
- Pull the original RDS repository: [https://github.com/mirjanastojilovic/RDS](url)
- Follow the instructions to set up the Basys3 experiment
- Insert the constraints file "frozen_sensor.xdc" from my repo.
- Use the program from the original RDS repo to generate traces: "./interface -k 1 -pt 1 -t [number_of_traces] -s -d [your_directory_name]". Do this once for a benign dataset and again for the test dataset.
- Run my script "cpa_sliding_window_integrity_checking.py" with arguments "--groupA" and "--groupB" specifying the benign and test dataset directories generated by the RDS program, respectively.
- Example: "python cpa_sliding_window_integrity_checking.py --groupA [path_to_directory_A] --groupB [path_to_directory_B]"
